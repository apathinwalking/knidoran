Elegaf (Elegant Game Framework) is a set of utilities written in javascript in order to provide the basics of a video game engine:
  • A Game Loop
  • A Key Listener
  • Sprite Handling
  • A Backbuffer
    • Although this is all Elegaf has currently, there is more to come. I am currently working on implementing Seidel's Triangulation Algorithm in order to implement collision detection. A demo of my triangulation algorithm will be available soon. The following is a basic demo of what Elegaf can do so far. It appears simple, but Elegaf is capable of much more. All the art was done by me, and is available for reuse upon request.
